# 图标资源目录

## 说明

此目录用于存放小程序的图标资源。

## 需要准备的图标

### TabBar 图标（必需）
所有图标建议尺寸为 81px * 81px（3倍图）

- `home.png` - 首页图标（未选中状态）
- `home-active.png` - 首页图标（选中状态）
- `health.png` - 健康图标（未选中状态）
- `health-active.png` - 健康图标（选中状态）
- `profile.png` - 我的图标（未选中状态）
- `profile-active.png` - 我的图标（选中状态）

### 其他图标
- `default-avatar.png` - 默认头像

## 图标设计建议

1. **尺寸规范**：
   - TabBar 图标：81px * 81px
   - 普通图标：根据实际使用场景确定

2. **格式要求**：
   - 支持 PNG、JPG 格式
   - 建议使用 PNG 格式（支持透明背景）

3. **颜色建议**：
   - 未选中状态：灰色系（#7A7E83）
   - 选中状态：主题色（#3cc51f）

## 使用图标资源

1. 从设计软件导出图标
2. 将图标文件放入此目录
3. 在 `app.json` 中引用图标路径

## 临时方案

如果暂时没有图标资源，可以：
1. 使用在线图标生成工具
2. 使用 emoji 表情代替
3. 使用纯色占位图
