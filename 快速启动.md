# 健康伙伴小程序 - 快速启动指南

## 🚀 快速启动步骤

### 步骤 1：打开微信开发者工具

确保已安装最新版本的微信开发者工具。

### 步骤 2：导入项目

1. 打开微信开发者工具
2. 点击 "+" 或 "导入项目"
3. 选择项目路径：`/Users/daimin/whpuedison/code/health-partner`
4. 输入 AppID（可以使用测试号）
5. 点击 "导入"

### 步骤 3：构建 npm 包

**重要：** 必须执行此步骤才能使用 Vant Weapp 组件库

1. 在微信开发者工具中，点击顶部菜单栏
2. 选择 "工具" -> "构建 npm"
3. 等待构建完成（会看到 "构建成功" 的提示）

### 步骤 4：编译运行

1. 点击 "编译" 按钮
2. 在模拟器中查看效果
3. 或使用真机预览

## 🎯 首次使用建议

### 1. 完善个人资料（必须）

点击 "健康" 标签 -> 编辑资料：
- 输入身高（例如：170 cm）
- 输入体重（例如：65 kg）
- 输入年龄（例如：25）
- 选择性别
- 输入体脂率（可选，例如：20%）

### 2. 设定健康目标

点击首页 "目标设置" 卡片：
- 设定目标体重
- 设定目标体脂率
- 设定目标完成日期
- 设定每日热量摄入
- 设定每日运动时长
- 设定每日饮水量

### 3. 开始记录

**记录饮食：**
点击首页 "饮食计划" -> 选择餐次 -> 输入食物和热量

**记录运动：**
点击首页 "运动计划" -> 选择运动类型 -> 输入时长和消耗热量

## 🔧 常见问题

### Q1: 构建 npm 失败？

**解决方案：**
1. 确认 `src/package.json` 文件存在
2. 在微信开发者工具的 "详情" -> "本地设置" 中
3. 勾选 "使用 npm 模块"
4. 重新执行 "工具" -> "构建 npm"

### Q2: 页面样式不正常？

**解决方案：**
1. 确认已执行 npm 构建
2. 检查 `src/miniprogram_npm/@vant/weapp/` 目录是否存在
3. 清除缓存后重新编译

### Q3: 数据没有保存？

**解决方案：**
- 在模拟器中，数据存储在本地
- 清除缓存会删除所有数据
- 确保填写完整信息后点击"保存"按钮

### Q4: TabBar 不显示？

**解决方案：**
1. 确认已构建 npm
2. 检查 `custom-tab-bar/` 目录文件是否完整
3. 重新编译项目
